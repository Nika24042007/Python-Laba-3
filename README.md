## Алгоритм решения
    1. Пользователь вводит строку команды в формате '<функция> [<список>] <доп.аргумент>'
    2. Если команда не является комендой окончания работы программа прасит строку
    3. Программа парсит строку ,превращая ее в список [функция, список из аргументов, доп.аргумент]
    4. Если команды нет в списке констант, то выдается ошибка, ели есть, то праграмма начинает выполнение функции
    5. Если команда была из мписка для работы с очередью проверит существование очереди, если очереди нет выдаст ошибку(если это была не команда создания очереди)
### Fibo/fibo_recursive
    - Выводит число фибоначи стоящие на n-ой позиции
    - Отсчет позиции начинается с единицы
    - Получает на вход целое, неотрицательное число
### Factorial/factorial_recursive
    - Выводит факториял n числа
    - Получате на вход целое неотрицательное число
### Bubble_sort
    - Стандартаная сортировка пузырьком
    - Выводит список a отсортированный по возрастанию
    - Получате на вход список из целых\вещественных числе или строк
### Bucket_sort
    - Карманная сортировка
    - Выводит список a отсортированный по возрастанию
    - Получате на вход список вещественных чисел в диапазоне от 0 до 1, а также не обязательный целочисленный неотрицательный аргумент buckets
### Counting_sort
    - Сортировка подсчетом
    - Выводит список a отсортированный по возрастанию
    - Получате на вход список из неотрицательных целых чисел
### Heap_sort
    - Сортировка с помощью пресбора кучи
    - Выводит список a отсортированный по возрастанию
    - Получате на вход список из целых\вещественных числе или строк
### Quick_sort
    - Быстрая сортировка
    - Выводит список a отсортированный по возрастанию
    - Получате на вход список из целых\вещественных числе или строк
### Radix_sort
    - Поразрядовая сортировка
    - Выводит список a отсортированный по возрастанию
    - Получате на вход список из неорицательных целых чисел и дополнительный аргумент base, обозначающий систему счисления(по умолчанию деситичная)
### Queue
    - Класс для работы с очередью на list
    - Содержит функции:
    -- Для создания очереди 
    -- is_empty проверка на пустоту
    -- len  длина очереди
    -- dequeue удалаение элемента
    -- enqueue добавление элементов
    -- front просмотр первого элемента

### Синтаксис
#### Fibo/fibo_recursive
    - Принимет только один аргумент
    - Аргумент должен быть целым неотрицетельным и ненулевым числом
#### Factorial/factorial_recursive
    - Принимает только один аргумент
    - Аргумент должен быть неотрицательным целым числом
#### Bubble_sort
    - Принимает только один аргумент
    - Аргумент должен быть списком из целых\вещественных чисел или строк(строки и числа не могут быть в одном списке)
#### Bucket_sort
    - Принимает один аргумент(обязательно), второй вариативно
    - Первый аргумент должен быть списком из чисел в диапозоне от 0 до 1, второй целеое, ненулевое и не отрицательное число (по умолчанию 10)
#### Counting_sort
    - Принимает только один аргумент
    - Аргумент должен быть списком из целых, неотрицательных чисел
#### Heap_sort
    - Принимет только один аргумент
    - Аргумент должен быть списком из целых\вещественных чисел или строк(строки и числа не могут быть в одном списке)
#### Quick_sort
    - Принимет только один аргумент
    - Аргумент должен быть списком из целых\вещественных чисел или строк(строки и числа не могут быть в одном списке)
#### Radix_sort
    - Принимате один аргумент(обязательно), второй аргумент вариабельно
    - Первый аргумент должен быть списком из неотрицательных целых чисел, второй целеое, ненулевое и не отрицательное число (по умолчанию 10)
#### Queue
    - принимет либо ничего либо один аргумент
    - Возвращает один аргумент либо None
